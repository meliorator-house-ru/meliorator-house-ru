## GitHub Pages
1. Создайте публичный репозиторий `meliorator-house`.
2. Загрузите содержимое архива в корень.
3. Settings → Pages.
4. Deploy from a branch.
5. Branch: main, Folder: /(root).
6. Save.


Дополнительно: мобильная версия переведена в единый стеклянный блок под заголовком, без свайпа и без нижней секции.
